# Helm Charts Collection

Репозиторий содержит коллекцию типовых Helm чартов для развертывания различных приложений в Kubernetes.

## Доступные чарты

- **[nestjs](./nestjs/)** - Универсальный чарт для развертывания NestJS приложений

## Использование

```bash
# Установка чарта
helm install my-app ./chart-name/

# Установка с кастомными значениями
helm install my-app ./chart-name/ -f values.yaml
```

## Требования

- Kubernetes 1.19+
- Helm 3.0+

Подробная документация по каждому чарту находится в соответствующей папке.
